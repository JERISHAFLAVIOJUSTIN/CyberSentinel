CyberSentinel

CyberSentinel is a Python-based cybersecurity log analyzer that detects suspicious login attempts from log files.

Features
Detects failed login attempts
Counts login failures by IP address
Calculates threat scores
Classifies IPs as LOW, MEDIUM, or HIGH risk
Technologies Used
Python
File Handling
Regular Expressions
Counter (collections)
Sample Input

2026-06-23 10:01:10 Failed login from 192.168.1.10

Sample Output

IP Address : 192.168.1.10

Failed Logins: 3

Threat Score : 30/100

Risk Level : MEDIUM
